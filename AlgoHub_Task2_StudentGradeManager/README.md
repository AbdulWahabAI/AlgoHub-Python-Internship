# AlgoHub Python Programming Internship - Week 2

## Project Overview
This is a simple Command Line Interface (CLI) application built to manage student records. It lets you enter a student name and roll number, add multiple courses with their marks, and automatically calculate their average score and final grade.

## Project Structure
* student_module.py: Contains the Student class which handles student data and grading logic.
* main.py: Contains the terminal menu loop that interacts with the user.
* requirements.txt: Lists the testing and code quality tools.

## How to Run the Project

1. Open your terminal inside the project folder.
2. Install the required development tools:
   pip install -r requirements.txt
3. Run the application using Python:
   python main.py

## How to Test Code Quality
To check the code format and quality before submitting, run these commands:
flake8 .
pytest
