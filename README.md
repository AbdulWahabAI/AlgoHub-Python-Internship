# week 1 console tasks

this folder has my two python scripts for week 1. everything runs from the main menu.

## features
- calculator: basic math functions (+, -, *, /) with error handling so it doesn't crash on division by zero.
- guessing game: random number game (1-100) with an exit feature if you want to quit early.

## running the app
open your terminal in this folder and type:
python main.py
